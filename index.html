What is this?

This is an HTML recreation of the lineup scene in the Marvel movie Guardians of the Galaxy. Rob Scanlon built this as a fun way to learn webgl fragment shaders. It was created using a bunch of open source libraries, most notably Three.js. Source code for this little project is available on Github.

Be sure to scroll down (or click on your down arrow, or swipe down if you are mobile) to see more widgets.

This is part 3 of a trilogy; check out Github Wargames and Encom Boardroom if you want to see what else I built while learning HTML5.
